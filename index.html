About

Student Finance Tracker

Know exactly where your money goes — and take it back.

What is this?

A lightweight, offline-first web app built for students to track daily spending, set budgets, and visualise financial habits — no accounts, no cloud, no cost.

Why it exists

Student budgets are tight. Small leaks — coffee, bus fare, impulsive snacks — add up fast. FinTrack gives you clarity with zero friction.

Key features

  • 6 default spending categories
  • Live regex-powered search
  • USD ↔ RWF ↔ NGN conversion
  • Budget cap alerts
  • JSON import / export
  • Works fully offline

Contact

Built as an academic project demonstrating regex, DOM manipulation, accessibility, and vanilla JS architecture.
